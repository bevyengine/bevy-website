If a player is to *play* your game, it has to be interactive.
Players will provide an action or a reaction, and the game will react and change in return.
To make this happen, we need to receive input from the player through some kind of device.
In Bevy, we can collect input from gamepads, keyboards, mice, or even touch screens.
The process of accessing and using this data is detailed on the [Using Input page](/learn/book/handling-input/using-input), which is where we'd recommend you start.
